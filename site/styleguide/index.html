Divider heading & subhead

.divider-heading: a mono, uppercase heading with a rule above it, for breaking a long document into major parts. .subhead is the smaller mono subheading beneath it, with an optional muted .note span inline.

Data table: availability matrix

.datatable.grid: a centered matrix with .rowlabel-col / .rowlabel and a colored .rail per row, td.c mark cells, and tr.group divider rows (one tbody per group) carrying a .glabel and .gdot. A .datalegend of .legend-swatch pills keys the marks; each cell pairs a .datamark with a .marklabel in a .markstack. The .gdot has no color in core - each page sets it inline from its own rail tokens, done here directly from core's status colors since this is a demo, not a real vendor row.
